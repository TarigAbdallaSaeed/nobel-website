# NOBEL Website

موقع ثابت احترافي وخفيف لمنظمة Network Organizations of Best Life - NOBEL.

## الملفات
- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`

## التشغيل

## تغيير اللغة
زر اللغة في أعلى الموقع يبدل بين العربية والإنجليزية، ويحفظ الاختيار في المتصفح باستخدام LocalStorage.
